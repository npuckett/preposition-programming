Concept

INTO means moving from outside to inside something, entering or penetrating a space, container, or boundary. This implies a transition from external to internal position.

Translation Strategy

To create "into" relationships on screen:

  • Boundary Crossing: Move an object from outside a defined area to inside it
  • Entry Points: Define specific entrance locations or general boundary crossing
  • State Transition: Track the change from "outside" to "inside" state

Key P5.js Methods

Boundary Detection:
• dist(x1, y1, x2, y2) - Calculate distance from boundary center
• For circular boundaries: distance < radius means inside
• For rectangular boundaries: check X and Y ranges

Movement Control:
• lerp(start, stop, amount) - Smooth movement toward interior
• Vector calculation for entry direction

Drawing Elements:
• ellipse(x, y, width, height) - Draw containers and moving objects
• rect(x, y, width, height) - Draw rectangular containers
• fill() with transparency to show containment

State Management:
• Boolean flags to track inside/outside status
• Entry detection and completion tracking

Mouse Interaction:
• mousePressed() - Trigger entry animation
• Button controls for animation

Code Structure

Basic Pattern for "Into":

1. Define container and entry point: Set up boundary and starting position
let container = {x: 200, y: 150, radius: 80};
let startPoint = {x: 50, y: 150};
let endPoint = {x: 200, y: 150}; // Inside container

2. Check containment: Detect inside/outside status
let distance = dist(object.x, object.y, container.x, container.y);
let isInside = distance < container.radius;

3. Animate entry: Move from outside to inside
if (isMoving && progress < 1) {
  progress += speed;
  object.x = lerp(startPoint.x, endPoint.x, progress);
  object.y = lerp(startPoint.y, endPoint.y, progress);
}

4. Track entry completion: Detect when fully inside
if (isInside && wasOutside) { hasEnteredContainer = true; }
