Concept

The preposition "during" indicates that something happens throughout the time span of another event or process. It describes continuous or ongoing activities that occur while something else is taking place. In this visualization, particles are emitted continuously during a traveler's journey from one point to another.

Visual Translation Strategy

To represent "during" relationships through movement:

  • Continuous Process: Actions that happen throughout an entire period
  • Journey-Based Events: Things that occur while traveling from A to B
  • Progressive Activity: Ongoing emission/creation during movement
  • Temporal Overlap: Secondary events happening within primary process

Key P5.js Methods

Movement and Timing:
• atan2() - calculates movement direction
• frameCount - controls emission timing during journey
• cos() and sin() - for directional movement
• random() - varies particle properties during emission

Particle System:
• push() - adds new particles during movement
• splice() - removes faded particles
• beginShape() - creates particle trails
• vertex() - builds trail paths

Visual Feedback:
• Alpha transparency for fading effects
• Progress tracking during journey
• Real-time particle count display

Code Structure

Basic Pattern for "During" Events:

1. Setup journey: Define start and end points
let traveler = {
  x: 50, y: 150, targetX: 350, targetY: 150,
  speed: 2, isMoving: false
};

2. Move during journey: Update position while traveling
if (traveler.isMoving) {
  let angle = atan2(dy, dx);
  traveler.x += cos(angle) * traveler.speed;
  traveler.y += sin(angle) * traveler.speed;
}

3. Emit particles during movement: Continuous creation
if (frameCount % 8 === 0) { // Every 8 frames
  let newParticle = {
    x: traveler.x, y: traveler.y,
    vx: random(-2, 2), vy: random(-2, 2)
  };
  particles.push(newParticle);
}

4. Update particles: Animate emitted particles
for (let particle of particles) {
  particle.x += particle.vx;
  particle.y += particle.vy;
  particle.life -= 0.01; // Fade over time
}
